## :zap: Powerable

### :memo: Description
Defines if the block send redstone power or not.

![Demo of powerable property](../../assets/properties/powerable.gif ':size=90%')

- [Tripwire Hook](https://minecraft.wiki/w/Tripwire_Hook).
